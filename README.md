# A3.2 · Redes Neuronales — Clasificación de Dígitos (MNIST)

Este proyecto implementa un modelo de **redes neuronales** para la **clasificación de dígitos del 0 al 9** utilizando la base de datos **MNIST**, tal como se realizó en el Taller de Redes Neuronales.  

El desarrollo incluye el entrenamiento, evaluación y mejora del modelo, así como la creación de un sistema que predice dígitos en **tiempo real** a partir de imágenes capturadas por cámara.

Se trabajaron los siguientes puntos principales:

- Entrenamiento y evaluación del modelo neuronal  
- Análisis de exactitud en entrenamiento, validación y prueba  
- Generación y preprocesamiento de imágenes propias  
- Evaluación del desempeño en imágenes externas  
- Implementación de mejoras y comparativas de resultados  
- Sistema de predicción en tiempo real
